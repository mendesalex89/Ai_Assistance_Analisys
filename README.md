# Formação Vibe Coding PCG

## Passo 1: Criar e Ativar o Ambiente Virtual

O ambiente virtual isola nosso projeto. Podemos criá-lo de duas formas: pedindo à IA ou executando o comando diretamente.

**Opção A: Pedir à IA**
```
Crie um ambiente virtual Python chamado .venv
```

**Opção B: Comando direto no terminal**
```bash
python -m venv .venv
```

Depois de criado, precisamos ativá-lo com o seguinte comando para que o terminal comece a usá-lo.

**Comando para ativar o ambiente (no Windows):**
```bash
.venv\Scripts\activate
```
